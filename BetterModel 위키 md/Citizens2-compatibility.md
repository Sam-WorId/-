BetterModel supports [Citizens2](https://www.spigotmc.org/resources/citizens.13811/) with some command.

## Sets a model of some NPC
```
/npc model <model> - sets some model to NPC
/npc model - remove model from NPC
```

## Animates a model of some NPC
```
/npc animate <id> <animation> [speed] [player] - Animates this model
```

## Animates some player NPC
```
/npc limb <id> <limb model> <animation> [player] - Animates this player
```
