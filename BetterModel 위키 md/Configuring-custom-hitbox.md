* **Step 1.** Add one of these bone tag to your group's name or set group's name as **hitbox**.  
```
b_
ob_
```
* **Step 2.** To define hitbox's size, create a element in this group and resize it as you want.
> [!TIP]\
> You can **always hide this element in BlockBench!**
* * *
![](https://github.com/user-attachments/assets/94aee9ed-9c2f-4975-92c4-3ea84ae31d24)  
* * *
You can make a multiple hitbox too.

> [!TIP]\
> Hitbox will **follow parent bone's animation.**
