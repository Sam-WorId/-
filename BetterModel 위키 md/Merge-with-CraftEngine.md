## Merge with [CraftEngine](https://polymart.org/product/7624/craftengine)

Since BetterModel's default path is already written in CraftEngine's default config, **you don't need to do something unless you edited BM's config**.

```yaml
  merge-external-zip-files:
    - "CustomNameplates/resourcepack.zip"
    - "BetterModel/build.zip"
```
