## Merge with [ItemsAdder](https://itemsadder.com/)
Note that you need ItemsAdder at least equals or more than version [4.0.4](https://github.com/PluginBugs/Issues-ItemsAdder/issues/3969).  
**Change BetterModel's build folder location to ItemsAdder's folder** in config.yml.

```yaml
build-folder-location: ItemsAdder/contents/BetterModel/resourcepack
pack-type: folder #BetterModel/config.yml
```
